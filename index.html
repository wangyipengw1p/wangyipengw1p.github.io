Research and Projects

Current
 [Thesis] DNN aided video decode chip based on RRAM @ Tsinghua
Area: Digital IC design, VLSI architecture, video compression

A software hardware codesign chip project for next generation video compression decoding. Optimising VCC by replacing block segmentation with NN method.

Mixed-Bit-Width Difference-Frame Sparse CNN Accelerator for Autopilot @ Tsinghua & Pi2star Tech
Area: Digital IC design, VLSI architecture, machine learning

At present, I've designed a mixed-bit-width sparse architecture for diff-processing in RTL and am analyzing the improvement using diff-frame processing. I aim to add reconfigurability later for such application that we need to switch between high performance and low energy mode.

Past
Depth map extraction using SpyNet and Sphere sweeping @ MICAS, KU Leuven
Area: FPGA design, VLSI architecture, machine learning, video (image) processing

We implemented depth map extraction for panoramic photo mosaic in both NN method and geomatric method. Acquired high accuracy by SpyNet and high through put in Sphere sweeping.

STICKER-IV: a universal neural network processor @ Pi2star Tech. [non-open source]

Embedded system(TX2) implementation of neural network-based object detection for drones
Hao Lu, Yipeng Wang, Yichao Sun, Yuhang Yuan, Yuanqing Cheng, Ying Wang
Area: Objection Dectecion, Network Utilization

We trained a network based on Yolov2 and ShuffleNet as a comprehensive solution for Objection detection on NVIDIA Tx2 platform. Our solution ranked 8th place in DAC-SDC 2019[Teamname: Ict-Jeejio-nobug]. I was in charge of the network training for Yolo and tridentNet(not used in final solution).

Phase noise(PN) compensation using dedicated pilot for mm-Wave OFDM with ASIC @ Lund U
Yipeng Wang, Minkeun Chuang, Liang Liu
Area: Massive MIMO, wireless communication, Digital design

We Implemented a hardware friendly algorithm for PN compensation using the dedicated pilot in 5G-OFDM. Through loosing the feedback loop and adding parallelism we achieved chieved the following implementation result: at least 500MHz for both 65nm balk and 28 nm ST FDSOI and at most 8 parallel; throughput reached at least 4 G complex num/s. I was in charge of the whole soft-hardware co-design design.
